=============================================================================================
OVERVIEW
	The starting interface to the Android Edition of a network-based Card game mobile app
	This release contains an interface that allows the user to find and create games as
	well as edit limited custom settings of the app. The application is designed to allow
	the user to create a game by editing custom settings that are both general to card
	games as a whole, as well as game specific settings and house rules. This release
	comes equipped with a limited version of this functionality specific to the games of
	Hearts and Euchre.
=============================================================================================
OPERATION
	1. Start:		The user can press the play button to look for available
				games(2) or choose to edit custom settings(6)

	2. Find Game:		The user is given a list of available local games which can
				be tapped to open a preview lobby(3) or the user is also
				given the option to create and host a new game(4)

	3. Preview Lobby:	The user is given a list of players currently in the game
				and a button which gives them the decision to join the
				viewed lobby(5)

	4. Create Game:		Allows the user to edit specific game settings such as the
				game type, the game title, as well as a game description to
				describe the game for users viewing the lobby. When the user
				is finished customizing game settings they can press the
				"Create Game" button to enter the lobby(5)

	5. Lobby:		Shows a list of players currently in the game and gives the
				game owner the option to start the game if an adequate number
				of players are available to play.

	6. Settings		Allows the user to edit custom game settings such as their
				display name.

	**Note that all displays may use the action bar to access the settings page**
